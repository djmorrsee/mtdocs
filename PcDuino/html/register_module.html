register_module module¶

This file contains functions for registering the pcduino and for removing it from the server.

This should only be done once

register_module.Register()[source]¶

This function lets the server know that there is a new module if the MOLUDULE_ID in constants.py is unique the server will return an authoraztion key that will be needed for any later api requests made by the pcduino. This key will be saved in a text file by this function. If the MODULE_ID is already in use the bad reqistraion staus code wil be printed

register_module.UnRegister()[source]¶

Sends an API requests to the server to remove all of data for this module from the database.

This should only be called if the module will be disconnected.
