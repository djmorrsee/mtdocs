take_reading module¶

This file is the main function that the pcduino will run. It will take a reading from the sensors and the send the data to the server. It will also send an API requests to delete all data older than 24 hours.

Function for getting an averaged reading from one of the sensor pins.

take_reading.Main()[source]¶

This function will take light and temperature raadings, format the data into JSON and send it our server. It will print the status code returned and will request that old data be deleted from the database
